# LifePlanner_Frontend

## Clone the Repository
```bash
git clone https://github.com/RahulGupta899/LifePlanner_Frontend
```

## Change directory
```bash
cd LifePlanner_Frontend
```

## Install Packages
```bash
npm install
```

## Run the Application
```bash
npm run dev
```
